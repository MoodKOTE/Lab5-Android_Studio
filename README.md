## Цель работы

Повторить основы разработки пользовательского интерфейса на Jetpack Compose, научиться работать с контейнерами `Row` и `Column`, изучить ARGB-модель представления цветов, освоить компонент `Scaffold` и базовые принципы Material Design.

---

## Краткое описание приложения

Разработано простое демонстрационное приложение, которое иллюстрирует:

- использование основных компоновщиков Compose: `Column`, `Row`, `Box`;
- создание пользовательских `@Composable` функций для переиспользования UI-компонентов;
- задание цветов в формате ARGB (шестнадцатеричное и десятичное представление) с различными уровнями прозрачности (Alpha-канал);
- применение модификаторов для управления размером, отступами и выравниванием;
- базовую структуру экрана с использованием `Scaffold` (верхняя панель, плавающая кнопка, контентная область).

### Функциональность экрана

На главном экране присутствуют:

1. **Заголовок** из двух строк текста, оформленный с помощью отдельной composable-функции.
2. **Горизонтальный ряд** из трёх цветных квадратов размером 100×100 dp:
   - первый квадрат — полностью непрозрачный синий (`0xFF0000FF`),
   - второй квадрат — полупрозрачный красный (`0x80FF0000`),
   - третий квадрат — зелёный, заданный десятичной ARGB (`Color(255, 0, 255, 0)`).
3. **Составной контейнер**: `Column` с двумя вложенными `Row`, внутри каждой `Row` выводится строка символов (буквы фильтруются), цвета строк различаются, добавлен вертикальный отступ с помощью `Spacer`.
4. Все цвета явно заданы в ARGB-формате.

Код приложения разбит на небольшие читаемые composable-функции, каждая отвечает за отдельную часть интерфейса (`TitleSection`, `ColorSquaresRow`, `SymbolsRow` и т.д.).

## Технологии

- Язык: **Kotlin**
- UI Toolkit: **Jetpack Compose**
- Минимальная версия SDK: **API 25**
- Библиотеки: **androidx.compose.material3**, **androidx.activity.compose**

## Запуск проекта

1. Клонируйте репозиторий.
2. Откройте проект в **Android Studio** (рекомендуется версия Hedgehog или более новая).
3. Дождитесь синхронизации Gradle.
4. Запустите приложение на эмуляторе или физическом устройстве с помощью `Shift+F10`.
